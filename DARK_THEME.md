# Pitch Black Dark Theme

The app now features a professional, classy pitch black dark theme inspired by premium code editors.

## Color Palette

### Background Colors
- **Primary Background**: `#000000` - Pure black for main areas
- **Secondary Background**: `#0a0a0a` - Slightly lighter for panels
- **Tertiary Background**: `#141414` - For cards and elevated surfaces
- **Hover Background**: `#1a1a1a` - Subtle hover states

### Text Colors
- **Primary Text**: `#ffffff` - Pure white for main content
- **Secondary Text**: `#a0a0a0` - Light gray for secondary info
- **Tertiary Text**: `#666666` - Darker gray for hints and placeholders

### Borders & Accents
- **Border Color**: `#222222` - Subtle borders that don't distract
- **Accent Color**: `#ffffff` - White for emphasis

### Interactive Elements
- **Button Background**: `#1a1a1a` - Dark gray base
- **Button Hover**: `#2a2a2a` - Lighter on hover
- **Button Active**: `#333333` - Even lighter when pressed
- **Input Background**: `#0f0f0f` - Very dark for inputs

### Scrollbars
- **Scrollbar Background**: `#0a0a0a`
- **Scrollbar Thumb**: `#333333`
- **Scrollbar Thumb Hover**: `#444444`

## Design Philosophy

### Monotone White Buttons
- Buttons use white text on dark backgrounds
- Active/selected buttons invert: white background with black text
- Clean, minimal aesthetic
- High contrast for accessibility

### Typography
- **Font**: Segoe UI for UI, Consolas for code
- **Weights**: Light to medium (400-500) for elegance
- **Letter Spacing**: Slightly increased for readability

### Spacing
- Generous padding for breathing room
- Consistent 1rem base unit
- Clean separation between sections

## Component Styling

### Header Bar
- Background: `#0a0a0a`
- Border: `#222222`
- Title: White, 400 weight, letter-spaced
- AI Button: Inverts when active (white bg, black text)

### Left Sidebar (Properties)
- Background: `#0a0a0a`
- Border: `#222222` on right
- Clean separation from center

### Center Area
- Background: Pure black `#000000`
- Split 50/50 for viewer and editor
- Thin border between: `#222222`

### Right Sidebar (AI Chat)
- Background: `#0a0a0a`
- Border: `#222222` on left
- Message bubbles:
  - User: White background, black text
  - AI: Dark gray `#141414` with white text
  - Code: Monospace font, white text

### Input Fields
- Background: `#0f0f0f` (very dark)
- Border: `#222222`
- Focus: `#666666` border
- Placeholder: `#666666` text

### Scrollbars
- Custom styled to match theme
- Dark background with lighter thumb
- Smooth hover transitions

## Visual Hierarchy

1. **Pure white** - Primary actions, active states, user messages
2. **Light gray** - Secondary text, labels
3. **Dark gray** - Borders, inactive states
4. **Pure black** - Main canvas, backgrounds

## Accessibility

- High contrast ratios (white on black)
- Clear focus indicators
- Readable font sizes (0.9rem - 1.1rem)
- Smooth transitions for state changes

## Professional Touch

- No bright colors or distractions
- Monochromatic elegance
- Clean lines and borders
- Subtle shadows removed
- Focus on content, not chrome

This theme creates a sophisticated, distraction-free environment perfect for coding and 3D modeling work.
